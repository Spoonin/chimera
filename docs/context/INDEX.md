- [2026-04-15] RedFlags table schema: severity, scope, message, status. *(telegram:@DisterX)*
- [2026-04-15] Snapshots table schema: as_of, equity_usd, realized_pnl, unrealized_pnl, total_pnl. *(telegram:@DisterX)*
- [2026-04-15] Positions table schema: market_slug, strategy_bucket, side, qty, avg_entry, mark_price, current_value_usd. *(telegram:@DisterX)*
- [2026-04-15] Allocations table schema: mandate_id, free_cash_usd, open_orders_usd, positions_cost_usd, equity_usd, pnl_usd. *(telegram:@DisterX)*
- [2026-04-15] Mandates table schema: mandate_id, token, nominal_usd, strategy_type, venue. *(telegram:@DisterX)*
- [2026-04-15] The database schema is Grist-ready and includes five tables: Mandates, Allocations, Positions, Snapshots, and RedFlags. *(telegram:@DisterX)*
- [2026-04-15] Realized and unrealized P&L must be tracked separately. *(telegram:@DisterX)*
- [2026-04-15] BANAL and PMM strategies cannot be mixed within the same market. *(telegram:@DisterX)*
- [2026-04-15] Every position must be tagged with a strategy_bucket of either BANAL or PMM. *(telegram:@DisterX)*
- [2026-04-15] P&L arises only from capital management, not from token issuance. *(telegram:@DisterX)*
- [2026-04-15] 1 token equals $1 USD obligation; token issuance does not create profit. *(telegram:@DisterX)*
- [2026-04-15] pnl_pct is calculated as: pnl_usd / nominal_obligation_usd × 100. *(telegram:@DisterX)*
- [2026-04-15] pnl_usd is calculated as: current_equity_usd − nominal_obligation_usd. *(telegram:@DisterX)*
- [2026-04-15] equity_usd is calculated as: free_cash_usd + reserved_collateral_usd + marked_positions_value_usd. *(telegram:@DisterX)*
- [2026-04-15] Accounting is structured across three layers: Obligations (capital entrusted per mandate), Allocation (where capital is sitting), and Positions (specific market entries). *(telegram:@DisterX)*
- [2026-04-15] PMM bucket performance is measured by spread capture, fill efficiency, and inventory risk. *(telegram:@DisterX)*
- [2026-04-15] BANAL bucket performance is measured by forecast quality and hypothesis selection. *(telegram:@DisterX)*
- [2026-04-15] Capital is organized into three trading buckets: BANAL, PMM, and MMXLM. *(telegram:@DisterX)*
