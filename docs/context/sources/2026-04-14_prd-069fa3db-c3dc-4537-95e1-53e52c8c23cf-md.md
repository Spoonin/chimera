## 1. Executive Summary
Chimera is an autonomous, agentic trading system built on the OpenClaw framework. - Autonomy: Operate 24/7 on a cloud-hosted OpenClaw instance (e.g., Tencent Lighthouse) with zero human intervention.

- Risk-Free Alpha: Achieve a positive Sharpe Ratio in a simulated environment using real-time price feeds.

## 3. System Architecture & Tech Stack
- Orchestrator: OpenClaw v0.4.x — manages skills, memory, and heartbeats
- Learning Engine: MetaClaw + OpenClaw-RL — intercepts trade results to update agent's internal policy
- Data Feed: CoinAPI / Alpaca MCP — real-time WebSocket feeds for Crypto and US Stocks
- Execution: Virtual Skill (Paper) — simulates order fills, slippage, and fees in a sandbox
- Memory: Milvus / Vector DB — stores historical market regimes and "lessons learned"

### 4.1 Meta-Strategy Research Agent
- Market Scanning via Exa Search Skill (financial news, sentiment shifts)

- Reward Signal: daily virtual P&L
- Policy Update: if strategy loses virtual capital, MetaClaw triggers "Post-Mortem" event

### 4.3 Market Making & Execution
- Bid/Ask Spread Management in high-volatility, low-liquidity virtual pairs
- Order Types: Market, Limit, Trailing-Stop via Alpaca Official MCP Server
